# Social Media Mastery - 12-Week Training Programme

A comprehensive, interactive web application for social media training covering Claude AI, Canva, Google Analytics, Google Ads, and Meta Ads.

## 🚀 Features

### Core Functionality
- **12-Week Structured Programme**: Progressive learning path with 60 interactive modules
- **AI-Powered Content**: Extensive Claude AI training with practical exercises
- **Design Skills**: Comprehensive Canva training for social media visuals
- **Analytics Mastery**: Deep dive into Google Analytics with real-world scenarios
- **Advertising Expertise**: Complete Google Ads and Meta Ads campaign management
- **Interactive Assessments**: 6 comprehensive quizzes with instant feedback
- **Progress Tracking**: Visual progress indicators and completion certificates

### User Experience
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Dark/Light Mode**: User preference with system detection
- **Offline Support**: Service worker enables offline functionality after initial load
- **Local Storage**: All progress saved locally, no server required
- **Search Functionality**: Global search across all resources and content
- **Bookmarking**: Save important resources for quick access

### Accessibility & Compliance
- **WCAG 2.1 AA Compliant**: Full accessibility support with screen readers
- **Keyboard Navigation**: Complete keyboard navigation support
- **Font Size Controls**: Adjustable text size for better readability
- **Reduced Motion Support**: Respects user motion preferences
- **Cross-Browser Compatible**: Works on Chrome, Firefox, Safari, Edge, and IE11

### Administrative Features
- **User Management**: Track multiple users and their progress
- **Comprehensive Reports**: Progress, assessment, and engagement analytics
- **Data Export**: Export user data and progress reports
- **Content Management**: Admin panel for content and assessment management

## 🛠 Technology Stack

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Custom styles with Tailwind CSS for rapid development
- **JavaScript (ES6+)**: Modern JavaScript with class-based architecture
- **Local Storage**: Client-side data persistence
- **Service Worker**: Offline functionality and caching
- **Font Awesome**: Icons and visual elements
- **No Dependencies**: Pure JavaScript implementation, no frameworks required

## 🆘 Support & Troubleshooting

### Common Issues
1. **Progress Not Saving**: Check browser local storage permissions
2. **Slow Performance**: Clear browser cache and reload
3. **Missing Content**: Ensure all JavaScript files are loaded
4. **Layout Issues**: Update to a supported browser version

### Getting Help
- Check browser console for error messages
- Verify all files are in the same directory
- Test in an incognito/private browsing window
- Try a different browser to isolate issues
